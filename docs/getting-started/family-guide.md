## When Something Isn't Working

### Quick Checklist

1. **Is it just you, or is everyone affected?**
   - Just you → Check your device (restart phone/computer)
   - Everyone → See "Who to Call" below

2. **Is it one thing or multiple things?**
   - One thing → Might be that specific service
   - Multiple things → Could be internet or main system

3. **Can you access the internet?**
   - No → Check your Wi-Fi, restart router if needed
   - Yes → Continue below

## Common Issues and Quick Fixes

### "I can't watch movies on Plex"

**Try this first:**
1. Close the Plex app completely
2. Wait 10 seconds
3. Open it again
4. If still not working → Call Todd

### "My smart lights aren't responding"

**Try this first:**
1. Check if your phone is on Wi-Fi (not cellular)
2. Try turning the lights off/on at the switch
3. Wait 30 seconds
4. Try again through Home Assistant
5. If still not working → Call Todd

### "The website won't load"

**Try this first:**
1. Check if other websites work
2. Try a different browser (Chrome, Safari, Firefox)
3. Clear your browser cache (ask Todd if you need help)
4. If still not working → Call Todd

### "My smart home app says 'connection error'"

**Try this first:**
1. Make sure you're on Wi-Fi
2. Close and reopen the app
3. Wait 1 minute
4. Try again
5. If still not working → Call Todd

## What NOT to Do

### Don't Try to Fix It Yourself
- **Don't** restart servers or routers (this can make things worse)
- **Don't** delete any files or folders
- **Don't** try to access anything labeled "admin" or "internal"
- **Don't** call anyone else for tech support without checking with Todd first

### Do Call Todd If
- Multiple family members can't access something
- Something that was working suddenly stopped
- You see any error messages with red text
- You're not sure what to do

## Emergency Situations

### If the Whole House Goes Dark (No Smart Home, No Internet, No Media)

1. **Stay calm** — Todd will fix it
2. **Call Todd immediately**
3. **Use regular lights** (there are switches everywhere)
4. **Use cellular data** on phones if Wi-Fi is down
5. **Wait for Todd's update** — he'll tell you when everything is back

### If You See a Fire or Water Leak Alarm

1. **This is a real emergency** — follow normal safety procedures
2. **Evacuate if needed**
3. **Call 911 first**, then call Todd
4. **Todd will handle the technical side** after everyone is safe

## Questions?

### "What does that mean?" — Common Terms

| Term | What It Means (Simple) |
|------|----------------------|
| **Server** | A computer that runs our services (like a movie server) |
| **Cloud** | Just means "somewhere on the internet" — we actually run our own |
| **Wi-Fi** | Wireless internet (the thing that makes your phone work without cables) |
| **Router** | The box that gives out Wi-Fi |
